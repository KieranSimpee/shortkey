## Shot formula (keeps brand clear)

1. Face + product in frame (0–3s)  
2. Say / show **brand name** on screen  
3. Apply / try-on result (shade match)  
4. End card: shop link / Shortkey CTRL shop  

Vertical 9:16 or 16:9 both work. Poster frames already wired in content if video is missing.
