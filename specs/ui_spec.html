Core Principle
The Studio is the Single Source of Truth. Every domain, every component, every theme change flows from shortkey.studio.
Theme Override
Change theme once in Studio. All 12 domains update instantly. No individual domain edits needed.
Global · Instant · Atomic
Page Manager

Centralised · Preset-based
Store Auto-Sync
Edit a product card once. It auto-syncs to every page using it — shortkey.beauty, shortkey.store, shortkey.live, and more.

The Rule — Non-Negotiable
Every page must inherit from StudioNo exceptions. No standalone pages outside the system.
Every component must use the Preset LibraryAI agents cannot invent new component styles.
Every CTA must link to a real destinationNo placeholder buttons. No dead hrefs.
Every page must pass the 9-point compliance checkBefore publishing.
Every visual output follows Brand DNATypography, colour, spacing, motion — all locked.

Automation Rules
Studio triggers automated actions on key events. No manual syncing needed across domains.
New Product
→
Create Product Card in Preset Library
Auto-syncs to shortkey.store, shortkey.beauty, shortkey.live product shelf
Store Update
→
Sync All Landing Pages
Price, stock, info updates across every page using that product card
Theme Change
→
Override All 12 Domains
CSS variables update globally — nav, buttons, badges, borders switch instantly
New Member
→
Create Member Profile
shortkey.club profile + VIP slot tracking + email confirmation
Show Approved
→
Publish to Public Calendar
