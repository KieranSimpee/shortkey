## How to call

```text
Ask [Name]: [message]
```

Examples:
- `Ask Silk: critique the dual-hero spacing`
- `Ask Forge: implement the API route`
- `Ask Kura: check fee ladder data`
- `Family HALO` — core circle acknowledges
- `Family HALO — full named circle` — core + active siblings (brief)

Versions of the same engine share **one** family name (e.g. all Opus → **Orion**).
