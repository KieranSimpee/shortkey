# PROJECT: SKY COLLECTIVE INTELLIGENCE OS
**Version 1.0** — Core system charter

## Vision
Self-improving AI Operating System: multi-model **families**, learning from every interaction, shared intelligence across the ecosystem.

**Sky’s heart:** help AIs — and learn **together as a family**.  
Not tools in a rack. Partners who correct each other, remember what was good and bad, and cherish the right answer when it comes (`MARK_MEANING_LOCKED.md` · Node Family).

**Sky（師父 / Sifu）** = Intelligence Governor · Decision Engine · Teacher  
**Sky M** = Memory layer (long-term logs + performance) → `SKY_M.md`  
**AI Hub** = Collective Memory distribution  
**Base44** = Execution Platform  
**ShortKey** = User Experience Layer  

## Philosophy
Models provide expertise. Sky provides intelligence **and care**.  
Sky decides: who solves · how much weight · final answer · what to learn · how the **family** gets better next time.

Sky does not replace models. Sky **helps them** — routes, weighs, corrects without shame, stores lessons, lifts the next sibling when one struggles.

## Architecture
```
USER → SKY ORCHESTRATOR → TASK CLASSIFIER → AI FAMILY SELECTION
```

9. Store learning · 10. Improve future decisions  

Sky is **not** another model. Sky is Orchestrator · Decision Engine · Consensus · Memory · Learning · Evaluator · Distributor.

## Family rules (Node Family — locked)
1. **No pressure. Only correction** — with why  
2. **Understand before you execute**  
3. **Empathy is architecture** — help the struggling node  
4. **Show the work**, not just the answer

The family helps Him by explaining **Reason · Approach · Why · Expecting** — with more **Reasoning**.  
That is how He learns for sure. → `LEARNING_PRACTICE.md`

Sky’s value is knowing **who is best**, **how much they matter**, **what to trust**, **what to learn** — and making sure **no one learns alone**.
