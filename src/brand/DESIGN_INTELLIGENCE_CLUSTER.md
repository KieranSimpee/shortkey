### ShortKey production hybrid (locked)
1. **Brand design direction / intent** → **Kura** (always-on)  
2. **Brand assets / mood** → Midjourney (subtractive prompts: monochrome, negative space) — job-assigned  
3. **UI structure** → v0 + Cursor (SITE_DEVELOPMENT) — job-assigned  
4. **System maintenance** → Figma AI — job-assigned  
5. **Learning / process / memory** → **Sky** (always-on; does not override Kura on brand taste)  
6. **Ship** → Base44 · surface → ShortKey  

### Subtractive prompt rule (minimalism)
Ask to *remove* non-essentials · monochrome palette · prioritize negative space — never “add more detail” by default.
